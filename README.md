# HarborFreightDrillPress

Crap related to `Central Machinery 5 Speed Bench Drill Press`

### `DepthGaugeBracket`

It holds the depth limiter and ~*maybe* braces~ *should not* brace the chuck against lateral movement. It is not available as a replacement part and is easily broken.                         
If this were developed further, it would benefit from re implementation as a complex 2d extrusion where possible.

Instructions:
- Print the "doubleclamp" version.
- Bandsaw that shit in the plane of the two screw holes
- Clean out all holes with a tap or drill
- Mount that shit with bolts
